# Swiggy Sample

A small React-based Swiggy clone demo using Parcel and Tailwind CSS.

## Project structure

- `index.html` — entry HTML file
- `script.js` — main React app bootstrap
- `src/components/` — React components
- `style.css` — Tailwind CSS import and base styles
- `package.json` — project dependencies and scripts

## Setup

```bash
npm install
npm start
```

Open `http://localhost:1234` in your browser.

## Build

```bash
npm run build
```

The production bundle is generated into `dist/`.

## Notes

- This project uses `react`, `react-dom`, `parcel`, and `tailwindcss`.
- The app fetches restaurant data from Swiggy's public listing API.
